# Habit Tracker Telegram Bot

Телеграм-бот для отслеживания полезных привычек.

## Возможности
- добавление и удаление привычек
- просмотр списка привычек
- отметка выполнения
- ежедневные напоминания
- статистика и streaks
- недельные отчёты
- достижения
- AI-советы по привычкам через API

## Структура проекта
- `handlers/` — обработчики команд и сообщений
- `keyboards/` — кнопки и меню
- `services/` — бизнес-логика
- `integrations/` — внешние API, включая AI
- `database/` — работа с БД
- `models/` — модели предметной области
- `utils/` — вспомогательные функции
- `tests/` — тесты
